Course Overview

The course aims to develop the fundamentals of analysis and design of controllers for linear dynamic systems, covering both continuous and discrete time. The primary focus is on multivariable systems (MIMO) and state-space representations.

Course Objectives

  • Master state-space analysis for multivariable systems.
  • Understand structural properties: Controllability and Observability.
  • Design advanced state feedback controllers and observers.
  • Implement robust control techniques using LMIs.

Summary Program

  • Linear Algebra & Matrix Analysis
  • State-Space Modeling & Stability
  • Controllability, Observability & Realization
  • State Feedback & Observer Design
  • Kalman Filtering & H2/H-infinity Control

Detailed Syllabus

1. Linear Algebra Review: Vector spaces, linear transformations, operators, and matrix decompositions.
2. State-Space Modeling: Representation of dynamic systems, canonical forms, Jordan forms, and similarity transformations.
3. Solution of State Equations: State transition matrix (matrix exponential) for time-invariant and time-varying systems.
4. Stability Analysis: Internal stability (Lyapunov) and external stability (BIBO).
5. Structural Properties: Controllability, reachability, observability, detectability, and stabilizability.
6. Controller & Observer Design: Pole placement via state feedback and design of state observers.
7. System Realization: Relationship between transfer functions and state-space representations.
8. Robust Control: Design of robust $\mathcal{H}_2$ and $\mathcal{H}_\infty$ controllers using Linear Matrix Inequalities (LMIs).
